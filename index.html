Connect Google Drive
Follow these steps once to enable Drive sync. Your data saves to health-tracker-data.json in your Drive.
1
Open Google Cloud Console
Go to console.cloud.google.com and sign in.
2
Create a project
Project dropdown - New Project - name it Health Tracker - Create.
3
Enable the Google Drive API
APIs and Services - Library - search Google Drive API - Enable.
4
Configure OAuth consent screen
APIs and Services - OAuth consent screen - External - fill in app name and email - Save and Continue through all steps.
5
Create OAuth Client ID
APIs and Services - Credentials - Create Credentials - OAuth client ID - Web application - add your GitHub Pages URL under Authorised JavaScript origins - Create.
6
Paste your Client ID below
Copy the Client ID ending in .apps.googleusercontent.com and paste it here.
